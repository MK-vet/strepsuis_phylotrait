# Test Reports

This directory contains test coverage reports and test artifacts.

## Contents

- `htmlcov/` - HTML coverage reports (generated by `pytest --cov-report=html`)
- `coverage.xml` - XML coverage report for CI integration
- `junit.xml` - JUnit-format test results

## Generating Reports

```bash
# Run tests with coverage
pytest --cov --cov-report=html --cov-report=xml

# Generate JUnit report
pytest --junitxml=test_reports/junit.xml
```

## Notes

- Report files are excluded from version control via `.gitignore`
